# O2Any 小红书使用指引（半自动）

本文档面向新手，介绍如何用 O2Any 生成小红书文案与素材包，再人工完成发布。

## 你能做到什么
- 生成适配小红书的纯文本文案（含标题、列表、引用等）。
- 一键导出“素材包”：文案 + 图片序列 + 上传顺序说明。
- 一键打开小红书创作平台页面。

> 说明：小红书暂无稳定的开放发布接口，本功能为“半自动导出 + 手动发布”流程。

## 使用前准备
- 确保当前笔记是 Markdown 文件（`.md`）。
- 图片建议为本地附件（Obsidian 附件管理），便于导出素材包。

## 使用步骤（桌面端）
1. 打开任意 Markdown 笔记。
2. 打开 O2Any 预览器（右侧面板）。
3. 在预览器顶部切换到「小红书」标签页。
4. 点击「刷新」生成文案预览。
5. 点击「复制文案」，文案将复制到剪贴板。
6. 点击「导出素材包」，会在当前笔记目录下生成导出文件夹。
7. 点击「打开小红书」，进入创作平台并粘贴文案、上传图片。

## 素材包导出位置与结构
导出路径位于当前笔记所在目录下：

```
小红书导出/文章标题-日期/
  - 文案.txt
  - 图片/
      01-xxx.png
      02-xxx.png
  - 上传顺序.txt
```

- `文案.txt`：最终可粘贴的文案。
- `图片/`：按顺序编号的图片。
- `上传顺序.txt`：每张图对应的顺序说明。

## 移动端使用说明
- 复制剪贴板在部分机型可能受限，如提示不支持，请手动选择文本复制。
- 如无法直接打开外部链接，请手动打开浏览器访问：
  - `https://creator.xiaohongshu.com/`

## 常见问题
### 1. 文案为空
- 请确认当前有激活的 Markdown 文件。
- 点击「刷新」后再尝试复制。

### 2. 图片未导出
- 图片需是本地附件或 Obsidian 能解析的链接。
- 若为外链图片，导出时可能提示未找到。

### 3. 排版与小红书显示不同
- 小红书编辑器会对格式进行二次处理，建议发布前在小红书内做微调。

## 建议写作规范（可选）
- 标题与小标题尽量简短。
- 列表用简洁的短句，利于移动端阅读。
- 图片配文放在对应图片附近，利于发布时对齐顺序。
